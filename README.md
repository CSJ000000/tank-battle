# 坦克大战 🎮

经典坦克大战游戏的 HTML5 实现，纯前端，无需服务器。

## 玩法

- **W/A/S/D** — 移动坦克
- **空格键** — 发射炮弹
- **P** — 暂停/继续

## 游戏特性

- 波次制敌方坦克生成，每波敌人数量递增
- 可破坏的砖墙和坚固的钢墙
- 敌人 AI 追踪玩家
- 击败敌人得分，生命归零游戏结束
- 复古像素风格渲染

## 部署到 GitHub Pages

1. 在仓库 Settings → Pages 中设置 Source 为 `main` 分支
2. 选择根目录 `/`
3. 访问 `https://CSJ000000.github.io/tank-battle/`
